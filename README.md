# flask-S3-test
## get started

```shell
git clone https://github.com/Kev1nXD/flask-S3-test
python3 -m venv venv
venv\Scripts\activate (on Windows)
source venv/bin/activate (on macOS)
pip install -r requirements.txt
aws configure (here you need input yor aws credentials: AWS Access Key ID, AWS Secret Access Key, Default region name)
SET S3_BUCKET=BUCKET_NAME (for Windows)
export VARIABLE_NAME=BUCKET_NAME (for Linux)
python.exe -m flask run 
```
